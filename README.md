# My Daily Articles

A personal blog where I write daily articles on various topics.

## Writing a New Article

1. Create a new file in the `_posts` directory
2. Name it: `YYYY-MM-DD-title-of-article.md`
3. Add front matter at the top (see example below)
4. Write your content in Markdown
5. Commit and push to GitHub

## Article Template
```yaml
---
layout: post
title: "Your Article Title"
date: YYYY-MM-DD HH:MM:SS +0000
categories: [category1, category2]
tags: [tag1, tag2]
---

Your article content here...
```

## Viewing Your Site

Your site will be available at: `https://snjwilson.github.io/blog/`
